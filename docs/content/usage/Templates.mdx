## Introduction

A workflow can be of be a type of instance or template. A workflow without a business defined is referred to as a template (link)

## Usage

### Create

#### Parameters

| Property | Type | Required | Example |
| :-- | :-- | :-: | :-- |
| id | Xuid | <CheckIcon color="green.5"/> | 'af6d8dfc-88b5-41f6-bbf5-825130390d64' |
| name | string | |  'template name (order)' |
| is_template | boolean | <CheckIcon color="green.5"/> | true |
| creator_id | Xuid | <CheckIcon color="green.5"/> | 'af6d8dfc-88b5-41f6-bbf5-825130390d64' |
| connection_id | Xuid | <CheckIcon color="green.5"/> | 'af6d8dfc-88b5-41f6-bbf5-825130390d64' |
| document_group_id | Xuid | <CheckIcon color="green.5"/> | 'af6d8dfc-88b5-41f6-bbf5-825130390d64' |
| owner_id | Xuid | <CheckIcon color="green.5"/> | 'af6d8dfc-88b5-41f6-bbf5-825130390d64' |

### Update

Update an existing Template.

#### Parameters

| Property | Type | Required | Example |
| :-- | :-- | :-: | :-- |
| id | Xuid | <CheckIcon color="green.5"/> | 'af6d8dfc-88b5-41f6-bbf5-825130390d64' |
| name | string | |  'template name (order)' | // Update template name
| is_template | boolean |  | true |
| creator_id | Xuid | | 'af6d8dfc-88b5-41f6-bbf5-825130390d64' |
| connection_id | Xuid | | 'af6d8dfc-88b5-41f6-bbf5-825130390d64' |
| document_group_id | Xuid | | 'af6d8dfc-88b5-41f6-bbf5-825130390d64' |
| owner_id | Xuid |  | 'af6d8dfc-88b5-41f6-bbf5-825130390d64' |

#### Usage

``` javascript
const template = await templatesService.update(
    {
        "_id": "c490e097-b232-4ee7-8958-2310a6060063",
        "name": "template name (order) v2",
        "is_template": 1,
        "creator_id": "fe615d30-05aa-448a-bd2a-0859816f0922",
        "connection_id": "4cd84112-bd8b-481b-adc8-3fd9483c3d71",
        "document_group_id" : "71aff4cc-9de2-4f4f-b140-d6d25982dd09",
        "owner_id": "8b3e0c15-65fd-4588-afd6-ae4fa0f40351",
    }
);
```

### Delete

Delete a Template. Response contains a deleted_at flag which signifies it has been deleted.

#### Parameters

| Property | Type | Required | Example |
| :-- | :-- | :-: | :-- |
| id | Xuid | <CheckIcon color="green.5"/> | 'af6d8dfc-88b5-41f6-bbf5-825130390d64' |

#### Usage

``` javascript
const template = await templatesService.delete('c490e097-b232-4ee7-8958-2310a6060063');
```

#### Response

``` json
   [
      "_id": "c490e097-b232-4ee7-8958-2310a6060063",
   ]
```
