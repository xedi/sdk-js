## Introduction

An Network Connection allows you to set up a connection between 2 mailboxes. Connection types include; FTP, XEDI, AS2.


## Usage

### list

Retrieve a list of NetworkConnections.

#### Usage

``` javascript
const networkConnections = await Xedi.NetworkConnections.list();
```

### get

Retrieve a networkConnection by Id.

#### Parameters

| Property | Type | Required | Example |
| :-- | :-- | :-: | :-- |
| id | Xuid | <CheckIcon color="green.5"/> | 'af6d8dfc-88b5-41f6-bbf5-825130390d64' |

#### Usage

``` javascript
const networkConnection = await Xedi.NetworkConnections.get('af6d8dfc-88b5-41f6-bbf5-825130390d64');
```

#### Response

``` json
{
   "_id": "af6d8dfc-88b5-41f6-bbf5-825130390d64",
   "partnership_id": "af6d8dfc-88b5-41f6-bbf5-825130390d64",
```

### Create

Create a new NetworkConnection

#### Parameters

| Property | Type | Required | Example |
| :-- | :-- | :-: | :-- |
| partnership_id | string | |  'hsahdhasd-88b5-41f6-bbf5-432423432re' |
| to_mailbox | string | |  'ddasdasd-88b5-41f6-bbf5-far3q42444' |
| from_mailbox | string |  |  'sfafafasd-88b5-41f6-bbf5-432432432' |
| network_status | Xuid |  | 'pending' |
| connection_type | Xuid | | 'FTP' |

``` javascript
const networkConnection = await Xedi.NetworkConnections.create(
```

### Delete

Delete a NetworkConnection. Response contains a deleted_at flag which signifies it has been deleted.

#### Parameters

| Property | Type | Required | Example |
| :-- | :-- | :-: | :-- |
| id | Xuid | <CheckIcon color="green.5"/> | '3n1jnj13-88b5-41f6-bbf5-m31kmk31mk1m31' |

#### Usage

``` javascript
const networkConnection = await Xedi.NetworkConnections.delete('3n1jnj13-88b5-41f6-bbf5-m31kmk31mk1m31');
```

#### Response

``` json
    {
        "_id": "3n1jnj13-88b5-41f6-bbf5-m31kmk31mk1m31"
    }
```
