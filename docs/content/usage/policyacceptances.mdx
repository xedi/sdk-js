## Introduction

The Policy Acceptances API allows you to interact with our Policy Acceptances data.

## Usage

### Create

Create a Policy Acceptance

**Parameters**

| Property      | Type      | Required                      | Example                                   |
| :--           | :--       | :-:                           | :--                                       |
| context_id    | String    | <CheckIcon color="green.5"/>  | 'a5a4acea-fbf9-11ea-9a4e-0100f1a50c6e'    |
| policy        | String    | <CheckIcon color="green.5"/>  | 'user_terms | business_terms'             |


**Usage**

```javascript
    await Xedi.policyAcceptances.create({
        "context_id": "a5a4acea-fbf9-11ea-9a4e-0100f1a50c6e",
        "policy": "business_terms"
    })
```
