## Introduction

The Transitions API gives the ability to fully configure each transition between the <Link to="/docs/usage/stages">stages</Link> of a work flow <Link to="/docs/usage/templates">template</Link>.

## Usage

### list

Retreive the list of transitions.

#### Usage 

``` javascript
const transitions = await Xedi.Transitions.list();
```

### get

Retrieve a transition by Id.

#### Parameters

| Property | Type | Required | Example |
| :-- | :-- | :-: | :-- |
| id | Xuid | <CheckIcon color="green.5"/> | '5076455a-a18e-11ea-8618-0242ac120002' |

#### Usage 

``` javascript
const transition = await Xedi.Transitions.get('5076455a-a18e-11ea-8618-0242ac120002');
```

#### Response

``` json
{
    "_id": "5076455a-a18e-11ea-8618-0242ac120002",
```

### create

Create a transition

#### Parameters

| Property | Type | Required | Example |
| :-- | :-- | :-: | :-- |
| from_id | Xuid |  | "9cabdeca-a0fe-11ea-8530-0242ac120002" |
| to_id | Xuid | <CheckIcon color="green.5"/> | "a099e2de-a0fe-11ea-b9be-0242ac120002" |
| workflow_id | Xuid | <CheckIcon color="green.5"/> | "9c82245e-a0fe-11ea-b1cf-01009cb3fa40" |
| description | string | | My first transition |
| is_automtic | bool | <CheckIcon color="green.5"/> | 1 |

#### Usage

``` javascript
const transition = await Xedi.Transitions.create(
  {
    from_id: '9cabdeca-a0fe-11ea-8530-0242ac120002',
    to_id: 'a099e2de-a0fe-11ea-b9be-0242ac120002',
    workflow_id: '9c82245e-a0fe-11ea-b1cf-01009cb3fa40',
    workflow_type: 'template',
    description: 'My first transition',
    is_automtic: 1
  }
);
```

#### Response

``` json
{
    "_id": "5076455a-a18e-11ea-8618-0242ac120002",
    "from_id": "9cabdeca-a0fe-11ea-8530-0242ac120002",
```
