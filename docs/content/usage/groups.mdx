## Introduction

The Groups API allows you to interact with our groups data

## Usage

### list

List all Groups that the user has the authority to see withing the context of a Business

**Parameters**

| Property | Type | Required | Example |
| :-- | :-- | :-: | :-- |
| pageNumber | number |  | 1 |

**Usage**

```javascript
const groups = await Xedi.Groups.list();
```

### Get

Retrieve a Group by ID that the user has the authority to see withing the context of a Business

**Parameters**

| Property | Type | Required | Example |
| :-- | :-- | :-: | :-- |
| id | Xuid | <CheckIcon color="green.5"/> | '96GE2a3e-boi0-11e9-b308-013533e48b37' |

**Usage**

```javascript
const group = await Xedi.Groups.get('96GE2a3e-boi0-11e9-b308-013533e48b37');
```

**Response**
```json
{
	"_id": "96GE2a3e-boi0-11e9-b308-013533e48b37",
	"name": "Team Cap",
	"owner": "c52abc50-b499-11ea-a48c-0100f1a50c6e",
	"group_type_id": "79fb3a3e-fcc0-11e9-b308-010060e48b37",
	"created_at": "2020-07-04 12:00:00",
	"updated_at": "2020-07-28 02:21:48"
}
```
