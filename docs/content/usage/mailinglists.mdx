## Introduction

The Mailinglists API allows you to interact with our Mailinglists data.

## Usage

### Create

Create a Mailinglist

**Parameters**

| Property      | Type      | Required                      | Example                                   |
| :--           | :--       | :-:                           | :--                                       |
| business_id   | Xuid      | <CheckIcon color="green.5"/>  | 'a5a4acea-fbf9-11ea-9a4e-0100f1a50c6e'    |
| name          | String    | <CheckIcon color="green.5"/>  | 'My List'                                 |
| description   | String    | <CheckIcon color="green.5"/>  | 'My new shiny Mailinglist'                |


**Usage**

```javascript
    await Xedi.Mailinglist.create({
        "businees_id": "a5a4acea-fbf9-11ea-9a4e-0100f1a50c6e",
        "name": "My Mailinglist",
        "description": "My Shiny new Mailinglist",
    })
```

### List

List All Mailinglists for a business

**Parameters**

| Property      | Type      | Required                      | Example                                   |
| :--           | :--       | :-:                           | :--                                       |
| businessXuid  | Xuid      | <CheckIcon color="green.5"/>  | 'a5a4acea-fbf9-11ea-9a4e-0100f1a50c6e'    |


**Usage**

```javascript
    await Xedi.Mailinglist.list("a5a4acea-fbf9-11ea-9a4e-0100f1a50c6e");
```

### Get

Get a mailinglist by id

**Parameters**

| Property         | Type      | Required                      | Example                                   |
| :--              | :--       | :-:                           | :--                                       |
| mailinglistXuid  | Xuid      | <CheckIcon color="green.5"/>  | 'e59631a8-fda6-11ea-a024-01006975ce16'    |


**Usage**

```javascript
    await Xedi.MailingLists.get("e59631a8-fda6-11ea-a024-01006975ce16");
```
